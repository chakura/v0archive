# V0 Archive — Claude Skills Pro Template Pack

このパックは、V0 Archive専用のClaude Project/Skillsを「3分割（Strategy / Studio Design / Creator Business）」で作るためのテンプレートです。

## 使い方（最短）
1. ClaudeでProjectを3つ作成
   - V0 Archive Strategy
   - Studio Design (V0 Archive)
   - Music Creator Business (V0 Archive)
2. 各Projectに Knowledge をアップロード
   - **V0-Core/** の全ファイル（共通コア）は3つ全部に入れる
   - それぞれの Skillフォルダ（Skill01/02/03）のファイルも対応するProjectに入れる
3. 各Projectの「Instructions」に、対応する INSTRUCTIONS_*.txt をコピペ

## アップロードのコツ
- Claudeはフォルダ階層を保持しない場合があるため、ファイル名先頭の番号（00/01/90…）で順序を固定しています。
- 図面、写真、見積、スプレッドシートは Assets/ を参考にして追加してください（ファイルは必要に応じて差し替え）。

## 最低限の“必須”ファイル
- V0-Core/01_CONCEPT_BRAND.md
- V0-Core/02_CUSTOMER_PERSONA.md
- Skill01_Strategy/05_UNIT_ECONOMICS.md
- Skill02_StudioDesign/04_SPACE_DESIGN.md
- Skill03_CreatorBusiness/40_ONBOARDING_SCRIPT.md

## 運用ルーティン（推奨）
- 週1回：V0-Core/00_INDEX.md を更新（KPI/課題/今週の結論）
- 重要決定のたび：V0-Core/93_DECISION_LOG.md に1行追加
- 不確定が出たら：V0-Core/92_ASSUMPTION_REGISTER.md に登録して期限を切る
