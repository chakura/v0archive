
# Unit Economics (Strategy)

## 入力（確定値/仮定を分ける）

### Zone2（ワークエリア）

- S1 = 6席（固定席数・確定）
- S2 = 9席（コワーキング席数・確定）
- M1 = 6名（固定席会員数・開業時目標・確定）
- M2 = 5名（コワーキング会員数・開業時目標・確定）
- D = 30日/月

### Zone1（スタジオ）

- H1 = 14h（1日提供可能時間・確定：9:00〜23:00）
- U_s = 0.30（稼働率・仮定）
- 無料枠 = 6h/月/名（固定席会員のスタジオ無料枠）

### 料金

- P1 = ¥100,000（固定席会費）
- P2 = ¥30,000（コワーキング会費）
- P_s = ¥5,000（スタジオ時間単価・通常昼間）

### コスト

- F1 = ¥400,000（家賃・仮定）
- F2 = ¥300,000（人件費・仮定：常駐1名・社会保険込み）
- F3 = F1 × 0.5（その他・仮定：家賃の5割）
- F = F1 + F2 + F3
- V = ¥80,000（変動費・仮定）
- CAPEX = ¥30,000,000（初期投資・確定）

### 追加売上

- Add = ¥0（未計上：コントロールデスク・レッスン・セッション）

---

## 容量

### Zone2

- 固定席 提供可能枠/月 = S1
- コワーキング 提供可能枠/月 = S2

### Zone1

- 提供可能時間/月 = H1 × D
- 固定席会員無料枠/月 = 6 × M1
- 実売可能時間/月 = (H1 × D) − (6 × M1)
- 実利用時間/月 = 実売可能時間 × U_s

---

## 売上モデル

**Zone2：A）会員課金**

- Zone2売上 = M1×P1 + M2×P2

**Zone1：B）稼働課金**

- Zone1売上 = 実売可能時間 × U_s × P_s

**合計売上 = Zone2売上 + Zone1売上 + Add**

---

## 利益

- 粗利 = 合計売上 − V
- 営業利益 = 粗利 − F

---

## 損益分岐

- 粗利率 = 粗利 / 合計売上
- 損益分岐売上 = F / 粗利率
- P_avg = (M1×P1 + M2×P2) / (M1+M2)
- 損益分岐会員数 = (F − Zone1売上) / P_avg

---

## 回収

- 回収期間（月）= CAPEX / 営業利益

---
