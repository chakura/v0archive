
# Glossary (言葉のブレを殺す)

* ブース：
  Zone1（スタジオ）内の個別録音・演奏空間。
  防音・音響処理が施された独立した部屋を指す。
  「スタジオ」と同義で使用可。
* 席：
  Zone2（ワークエリア）内の作業スペース。
  固定席（専用個室席）のみ。
  「ブース」とは混用しない。
* 会員：
  V0 Archiveに登録した利用者の総称。
  以下の2種類に分類する：
  1. 固定席会員（Core）：月額¥100,000・専用個室席
  2. スタジオ会員（External）：登録無料・スタジオのみ利用可
     ※「お客様」という呼称は使わない。
* 体験：
  以下の段階を明確に区別する：
  1. 見学：無料・予約制。空間と思想を確認する場。
     「自分がここに属するかを確かめる」文脈で設計する。
  2. トライアル：現時点では設定しない。
     ※「お試し」「気軽に」という文脈では使わない。
* 稼働率：
  Zone1とZone2で定義が異なる。計算式は91_KPI_DEFINITIONS.md参照。
  Zone1：実利用時間 / 実売可能時間
  Zone2：実利用時間 / 提供可能時間
  ※「混雑率」「埋まり具合」とは混用しない。
