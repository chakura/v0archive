## 3. 重要論点

### 3-1. 天井高 CH=2,760mm とBox-in-box（確定方針）

**Box-in-box構法は非交渉。CH=2,760mmであっても実施する。**
生ドラム演奏がZone1の必須要件であり、Box-in-boxを省略する選択肢はない。

オーナーが常用するスタジオの天井高は約2,300mm。
→ 有効天井高2,300mm台でもドラム演奏は成立するという**実体験ベースの裏付け**がある。

#### Box-in-box有効高の試算

| スラブ–スラブ高 | 浮き床（100〜200mm） | 吊り天井（150〜300mm） | 有効天井高        | 評価                                    |
| ---------------- | -------------------- | ---------------------- | ----------------- | --------------------------------------- |
| 3,000mm          | −150mm              | −200mm                | **2,650mm** | 余裕あり                                |
| 2,760mm          | −150mm              | −200mm                | **2,410mm** | 実用上問題なし（常用スタジオ≒2,300mm） |
| 2,760mm          | −200mm              | −300mm                | **2,260mm** | 最低限だが成立（常用スタジオと同等）    |

→ CH=2,760mmがスラブ間だった場合でも、浮き床・吊り天井の仕様を薄型に寄せれば**有効高2,300〜2,400mm**は確保可能。
→ 施工業者との打ち合わせでは「有効高2,300mm以上確保」を必須条件として提示する。

### 3-2. Zone2 寸法制約とパーティション構成の分岐

#### 東西（A–B）

Zone2の東西有効幅は**A–B**で統一される。

* 北壁ライン：B-C-D-Eの**柱の出っ張り**があるため、A–Bが上限
* 南壁ライン：T–Sにはドアがあり塞げないため、ブース配置の有効幅はA–Bと同等
* → **北列・南列ともA–Bの幅に収める必要がある。**

ブース東西合計（P1×1 + P2×3）= 2,100 + 1,700×3 = **7,200mm**

| A–B実測値     | パーティション構成                         | 判定              |
| -------------- | ------------------------------------------ | ----------------- |
| ≥7,600mm      | 標準（114mm×3）= 342mm追加 → 合計7,542mm | ✅ 現行プラン成立 |
| 7,500〜7,599mm | 薄型（81mm×3）= 243mm追加 → 7,443mm      | ⚠️ 薄型で対処   |
| <7,500mm       | 薄型構成（81mm）で対処                     | ⚠️ 薄型確定     |

**固定席は8席（4+4配置）で確定。** A–Bの値によりパーティション構成（標準/薄型）を決定する。

#### 南北（A–T）

ブース奥行D1,600 × 2列 + 通路800mm + パーティション厚 × 2 が必要幅。

| パーティション構成 | 必要A–T幅 | A–T=4,400mm時の判定  |
| ------------------ | ---------- | --------------------- |
| 標準（114mm×2）   | 4,428mm    | ⚠️ 28mm不足の可能性 |
| 薄型（81mm×2）    | 4,362mm    | ✅ 38mm余裕で成立     |

→ A–T実測値が4,400mmであれば **薄型構成で確実に成立** 。
→ A–Tが4,430mm以上あれば標準構成も成立。

#### フォールバック仕様（A–Bが大幅不足の場合のみ）

| プラン              | ブース内寸       | デスク         | 席数          |
| ------------------- | ---------------- | -------------- | ------------- |
| プラン1（ラック型） | W2,100 × D1,600 | W1,400 × D600 | **2席** |
| プラン2（デスク型） | W1,700 × D1,600 | W1,600 × D700 | **6席** |

各列：1×W2,100 + 3×W1,700 = **7,200mm** → 標準パーティション込みでも7,542mm。
A–Bが7,200mm以上あれば8席は何らかの形で成立する。

### 3-3. 吹抜（幅1,075mm）の影響

吹抜は**共用部扱い**であり、専有面積101.38㎡には含まれない（階段・サニタリーと同じ）。
→ ロビー有効面積への直接的な影響はなし。

残る確認事項：

* 吹抜がテナント空間に隣接する場合、**音の伝播経路**になる可能性（上下階へ）
* 吹抜周辺の壁仕様（遮音性）を現調で確認

### 3-4. Zone1レイアウト（確定方針）

#### CR＋出入口の制約寸法 = G–O

CRと出入口はZone1の南西コーナーに配置する。
このスペースの南北幅は**G–O**の距離で決まる。

| G–O実測値     | CR＋出入口の成否                                             |
| -------------- | ------------------------------------------------------------ |
| ≥4,000mm      | CR（≒2,500mm）＋出入口二重扉（≒1,500mm）が余裕をもって成立 |
| 3,000〜3,999mm | CR幅を圧縮して成立可能。デスクW1,400×D600は収まる           |
| <3,000mm       | CR＋二重扉の同時配置が困難。レイアウト再検討                 |

#### CRの設計要件

* 1人用。デスクW1,400×D600が1台入ればよい
* 簡易防音（Box-in-boxではない）。Zone1からの音漏れは許容
* スタジオとの間にモニター窓（ガラス）を設置
* CRの存在自体がBox-in-box壁〜RC躯体壁の間の**空気緩衝層**として機能する
* 防音の多重構造：Box-in-box壁 → CR空間 → CR壁 → ロビー
* 内装・施工はotokura株式会社に一任

#### このレイアウトの利点

1. スタジオ面積の最大化（南北フルスパン → 20畳確保）
2. CRが防音の緩衝帯として機能（構造的メリット）
3. 二重扉がCR動線上に自然に組み込まれる（サウンドロック）
4. CRを録音専用に限定することで設備・面積を最小化
5. ミキシングをZone2に分離することで、スタジオ稼働率とCR占有時間が競合しない
6. 前室が避難経路と倉庫を兼ねることで面積ロスなし
