## 3. 重要論点

### 3-1. 天井高 CH=2,760mm とBox-in-box（確定方針）

**Box-in-box構法は非交渉。CH=2,760mmであっても実施する。**
生ドラム演奏がZone1の必須要件であり、Box-in-boxを省略する選択肢はない。

オーナーが常用するスタジオの天井高は約2,300mm。
→ 有効天井高2,300mm台でもドラム演奏は成立するという**実体験ベースの裏付け**がある。

#### Box-in-box有効高の試算

| スラブ–スラブ高 | 浮き床（100〜200mm） | 吊り天井（150〜300mm） | 有効天井高        | 評価                                    |
| ---------------- | -------------------- | ---------------------- | ----------------- | --------------------------------------- |
| 3,000mm          | −150mm              | −200mm                | **2,650mm** | 余裕あり                                |
| 2,760mm          | −150mm              | −200mm                | **2,410mm** | 実用上問題なし（常用スタジオ≒2,300mm） |
| 2,760mm          | −200mm              | −300mm                | **2,260mm** | 最低限だが成立（常用スタジオと同等）    |

→ CH=2,760mmがスラブ間だった場合でも、浮き床・吊り天井の仕様を薄型に寄せれば**有効高2,300〜2,400mm**は確保可能。
→ 施工業者との打ち合わせでは「有効高2,300mm以上確保」を必須条件として提示する。

### 3-2. Zone2東西有効幅 = A–B（不足リスク）

Zone2の東西有効幅は**A–B**で統一される。

* 北壁ライン：B-C-D-Eの**柱の出っ張り**があるため、A–Bが上限
* 南壁ライン：T–Sにはドアがあり塞げないため、ブース配置の有効幅はA–Bと同等

→ **北列・南列ともA–Bの幅に収める必要がある。**

* 図面推定値：柱芯7,750mm − 壁厚300mm − 柱の出っ張り分 = **A–Bは7,450mm以下の可能性**
* 04_SPACE_DESIGNの要求値： **≥7,600mm** （現行プラン4+4・8席の成立条件）
* → **不足する可能性がある。**

#### 対処方針

| A–B実測値 | 判定                                                          |
| ---------- | ------------------------------------------------------------- |
| ≥7,600mm  | 現行プラン（P1 W2000×4席 / P2 W1600×4席）で4+4成立          |
| <7,600mm   | **フォールバック：P1 W2000×2席 / P2 W1600×6席に変更** |

#### フォールバック仕様

| プラン              | ブース内寸       | デスク         | 席数          |
| ------------------- | ---------------- | -------------- | ------------- |
| プラン1（ラック型） | W2,000 × D1,600 | W1,400 × D600 | **2席** |
| プラン2（デスク型） | W1,600 × D1,600 | W1,600 × D700 | **6席** |

#### フォールバック時の必要東西幅

各列：1×W2000 + 3×W1600 = **6,800mm** → 確実に成立。

**固定席は8席（4+4配置）で確定。** A–Bが不足した場合はフォールバック仕様で4+4を維持する。

### 3-3. 吹抜（幅1,075mm）の影響

吹抜は**共用部扱い**であり、専有面積101.38㎡には含まれない（階段・サニタリーと同じ）。
→ ロビー有効面積への直接的な影響はなし。

残る確認事項：

* 吹抜がテナント空間に隣接する場合、**音の伝播経路**になる可能性（上下階へ）
* 吹抜周辺の壁仕様（遮音性）を現調で確認

### 3-4. Zone1レイアウト（確定方針）

#### CR＋出入口の制約寸法 = G–O

CRと出入口はZone1の南西コーナーに配置する。
このスペースの南北幅は**G–O**の距離で決まる。

| G–O実測値     | CR＋出入口の成否                                             |
| -------------- | ------------------------------------------------------------ |
| ≥4,000mm      | CR（≒2,500mm）＋出入口二重扉（≒1,500mm）が余裕をもって成立 |
| 3,000〜3,999mm | CR幅を圧縮して成立可能。デスクW1400×D600は収まる            |
| <3,000mm       | CR＋二重扉の同時配置が困難。レイアウト再検討                 |

#### 構成

CRと出入口を**南西コーナーに集約**し、スタジオを南北フルスパンに拡張する。

```
┌──────────────────────────────────┐
│          スタジオ（Box-in-box）     │ ← 南北フルスパン
│          ≒33.4㎡（約20.2畳）       │
│                                    │
│                                    │
├────────┤                           │
│ 出入口  │                           │
│（二重扉）│                           │
├────────┤                           │
│  CR     │                           │
│ ≒3.8㎡ │                           │
└────────┴───────────────────────────┘
  ↑ロビー側
```

#### 面積配分

| エリア                   | 東西      | 南北      | 面積                           | 備考                        |
| ------------------------ | --------- | --------- | ------------------------------ | --------------------------- |
| スタジオ（Box-in-box内） | ≒5,100mm | ≒6,551mm | **≒33.4㎡（約20.2畳）** | 目標20畳に到達              |
| CR（簡易防音）           | ≒1,500mm | ≒2,500mm | **≒3.8㎡（約2.3畳）**   | 1人用・録音コントロール専用 |
| 出入口（二重扉＋前室）   | ≒1,500mm | ≒1,500mm | **≒2.3㎡**              | サウンドロック機能          |

※ 南北7,151mmは仮定値。現地実測で確定要。

#### 役割分担（確定）

| 機能                       | 場所                  | 備考                                                 |
| -------------------------- | --------------------- | ---------------------------------------------------- |
| **録音コントロール** | CR（Zone1内）         | 1人用。デスクW1400×D600のみ。録音オペレーション専用 |
| **ミキシング**       | 各自の固定席（Zone2） | CRではやらない。各会員が自席で完結する               |

#### CRの設計要件

* 1人用。デスクW1400×D600が1台入ればよい
* 簡易防音（Box-in-boxではない）
* スタジオとの間にモニター窓（ガラス）を設置
* CRの存在自体がBox-in-box壁〜RC躯体壁の間の**空気緩衝層**として機能する
* 防音の多重構造：Box-in-box壁 → CR空間 → CR壁 → ロビー

#### このレイアウトの利点

1. スタジオ面積の最大化（南北フルスパン → 20畳確保）
2. CRが防音の緩衝帯として機能（構造的メリット）
3. 二重扉がCR動線上に自然に組み込まれる（サウンドロック）
4. CRを録音専用に限定することで設備・面積を最小化
5. ミキシングをZone2に分離することで、スタジオ稼働率とCR占有時間が競合しない

## 4. 現調サマリー：何がわかっていて、何を測るか

### 図面からわかっている（推定値あり・実測で確定要）

| 距離 | 推定値        | 根拠                              |
| ---- | ------------- | --------------------------------- |
| A–B | ≒7,450mm以下 | 柱芯7,750mm − 壁厚 − 柱出っ張り |
| I–J | ≒6,650mm     | 柱芯6,950mm − 壁厚300mm          |
| J–K | ≒6,851mm     | 柱芯7,151mm − 壁厚300mm          |

※ すべて壁厚150mm仮定での推定。壁厚の実測で精度が変わる。

### 図面からわからない（推定値なし・実測必須）

| 距離                       | 何が決まるか                                                           |
| -------------------------- | ---------------------------------------------------------------------- |
| **A–T**             | Zone2の南北幅。ブース奥行D1,600×2列＋通路800mm（≥4,200mm必要）の成否 |
| **G–O**             | CR＋出入口のスペース。≥4,000mmで余裕、<3,000mmでレイアウト再検討      |
| **スラブ–スラブ高** | Box-in-boxの有効天井高。有効高2,300mm以上が必須条件                    |

### 結論：現調で測ってくる7項目

| # | 距離                           | 何が決まるか                           |
| - | ------------------------------ | -------------------------------------- |
| 1 | **A–B**                 | Zone2東西有効幅 → 8席の配置判定       |
| 2 | **A–T**                 | Zone2南北内法 → ブース2列＋通路の成否 |
| 3 | **I–J**                 | Zone1東西内法 → スタジオ面積の確定    |
| 4 | **J–K**                 | Zone1南北内法 → スタジオ20畳の成否    |
| 5 | **G–O**                 | CR＋出入口の南北幅 → レイアウト成否   |
| 6 | **スラブ–スラブ高**     | Box-in-box有効天井高                   |
| 7 | **壁厚** （開口部1箇所） | 上記すべての推定値の精度確定           |

**この7つがあれば、8席の配置・スタジオ20畳・CR配置・Box-in-box仕様、全部決まる。**
